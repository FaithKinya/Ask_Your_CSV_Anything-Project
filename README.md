# LLM-Powered CSV Analytics Assistant

This project is a Streamlit dashboard that lets users ask natural language questions
about the Superstore dataset. It uses a hybrid approach:

- Rule-based intent extraction
- Pandas aggregation for analytics
- Optional integration with LLM for more complex queries

### Features
- Total sales, sales per region, sales per year
- Most profitable categories
- Top-selling products
- Fully interactive Streamlit dashboard
